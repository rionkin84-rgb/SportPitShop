# SportPitShop

Информационная система интернет-магазина спортивного питания.

## Описание

Проект разработан в рамках учебной практики по специальности  
09.02.07 "Информационные системы и программирование".

## Функционал

- Авторизация пользователей
- Каталог товаров
- Поиск и фильтрация
- Корзина
- Оформление заказа
- Управление заказами
- Разграничение прав доступа

## Технологии

- C#
- .NET Framework 4.8
- Windows Forms
- Microsoft SQL Server
- Visual Studio 2022

## База данных

База данных содержит 11 таблиц:
- Role
- User
- Product
- Category
- Unit
- Producer
- Provider
- Order
- ProductInOrder
- PickUpPoint
- OrderStatus
